Intel Celeron J1900
===================

Intel\ :sup:`速` Celeron\ :sup:`速` J1900 quad-core CPU operates at 2GHz and supports up to 8MB of low voltage DDR RAM (DDR3L 1333).
Its Thermal Design Power (TDP) of 10W makes it an excellent choice for low-power mobile, handheld, and embedded applications. It also features Intel\ :sup:`速` 
HD Graphics for Intel Atom\ :sup:`速` Processor Z3700 Series, delivering excellent 2D/3D acceleration performance and HD media capabilities.
